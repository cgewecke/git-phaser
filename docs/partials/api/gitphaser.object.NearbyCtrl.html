NearbyCtrl
object in module gitphaser

Description

Controller for the nearby route. Exposes Meteor mongo 'connections' to DOM, filtered against current user as 'transmitter'. Subscription to 'connections' is handled in the route resolve. Also exposes GeoLocate service (for the maps view) and Notify service (to trigger notification when user clicks on list item to see profile)

Properties

  • listSlide

    Number: Slide constant 0. If slide === listSlide, the list view is shown.

  • mapSlide

    Number: Slide constant 1. If slide === mapSlide, the map view is shown.

  • slide

    Number: Slide variable. Set to 0 or 1 to show list/map view respectively
