Beacons
service in module gitphaser

Description

Service that transmits and receives iBeacon signal.

Methods

  • getUUID(index)

    Exposes the uuid array. When the user is creating an account by logging in for the first time the modulus of the server-generated Beacon minor and the uuid array length is used to select a uuid. This allows them to be distributed evenly across acounts and minimizes the likelyhood that a duplicate uuid will be present in any group of phones.

    Parameters
    ParamTypeDetails
    indexnumber

    The index of the uuid array to select

  • initialize()

    Sets up beaconing in app. This method resolves on the Nearby tab, so it may have already run as user navigates around.

    Returns
    promise

    Rejects if user does not authorize background beacon use

Properties

  • initialized

    Boolean state: true when user has authorized background beacon use, false otherwise.

  • quantity

    Number of available beacon uuids to select a broadcast user id from

  • regions

    Array of beacon region objects the service monitors for.
