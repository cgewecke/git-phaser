beaconMap

Description

<beacon-map> wraps a MapBox map that shows the user's current location. Loads or updates the map when the map slide is selected.

Usage

as attribute
<ANY beacon-map
     [slide="{number}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
slide
(optional)
number

The current slide value. If slide's val is '1', the map is either initialized or updated.
