hireable

Description

<hireable> Cash icon visible if attr 'available' is true. Tapping icon shows a brief toast with message: {{name}} is available for hire

Usage

as attribute
<ANY hireable="{boolean}"
     [name="{string}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
hireable
(optional)
boolean

Value of hireable key in the github user object

name
(optional)
string

Value of the login key of an github account.info object
