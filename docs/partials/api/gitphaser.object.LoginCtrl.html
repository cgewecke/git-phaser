LoginCtrl
object in module gitphaser

Description

Controller for the login route. Functions/Methods for a two-step login process which signs user into GitHub and then Meteor using details from a GitHub profile

Methods

  • devLogin()

    Bypasses authentication call which cannot run in browser because cordova inAppBrowser is device/simulator only

  • login()

    Authenticates with GitHub, loads GitHub profile and passes to meteor login handlers. Shows toast on authentication failure.
