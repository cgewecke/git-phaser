GeoLocate
service in module gitphaser

Description

Provides geolocation, reverse geocoding, and map display

Methods

  • getAddress()

    Gets device coordinates and sets public vars 'lat', 'lng'. Reverse geocodes coordinates and sets public var 'address'. Error states set lat, long and address to 0,0,'' respectively.

    Returns
    promise

    Resolves either an address string or ''.

  • isEnabled()

    Runs in the nearby route resolve block to trigger permissions, detect whether geolocation is enabled.

  • loadMap()

    Loads a Leaflet map with MapBox titles using devices current coordinate

  • updateMap()

    Resets map view and marker to current device coordinates

Properties

  • address

    User's current location expressed as address (String)

  • enabled

    Service state, determined by user authorizing geolocation on device (Boolean)

  • lat

    User's current latitude (Float)

  • lng

    User's current longitude (Float)

  • map

    Leaflet map object

  • map

    Leaflet marker object
