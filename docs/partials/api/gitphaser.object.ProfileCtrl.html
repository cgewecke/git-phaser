ProfileCtrl
object in module gitphaser

Description

Exposes GitHub.me profile object or account object to the profile template. Governs the /tab/profile and others/:username routes.

Methods

  • back()

    Navigates to tab.nearby. For nav back arrow visible in the others route.

  • follow()

    Wraps GitHub.follow and hides the follow button when clicked.

Properties

  • modalOpen

    Boolean: Triggers appearance changes in the template when a contact modal opens. See 'contact' directive.
