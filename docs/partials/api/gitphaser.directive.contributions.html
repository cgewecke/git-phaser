contributions
directive in module gitphaser

Description

<contributions> A Github contributions graph scraped from an svg resource. This has to be obtained by server proxy. The directive shows a spinner until the svg is received. SVG is embedded in a lateral scroll container that gets moved right-wards the maximum amount on load.

Usage

as element:
<contributions
       [name="{String}"]>
</contributions>

Parameters

ParamTypeDetails
name
(optional)
String

The login key of an github account.info object
